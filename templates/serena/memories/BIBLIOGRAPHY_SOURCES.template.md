## Adding New References

### Entry Format
```bibtex
@article{authorYYYYkeyword,
  author = {Last, First},
  title = {Title Here},
  journal = {Journal Name},
  year = {YYYY}
}
```

### Citation Key Convention
Format: `authorYYYYkeyword`
- Example: `clark1998extended`

### Checklist for New Sources
1. [ ] Verify citation key follows convention
2. [ ] Place in correct thematic section
3. [ ] Add annotation to this memory file
4. [ ] Note "When to cite" guidance
